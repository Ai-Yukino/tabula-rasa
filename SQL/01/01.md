## Question 3

SELECT everything from the songs table whose id numbers are even and greater than 50, or odd and less than ten.

### Solution

```
SELECT * from songs WHERE
(id % 2 = 0 AND id > 50) OR
(id % 2 = 1 AND id < 10)
```
